# 🚀 MoodMate – Premium AI-Powered Mood Tracker & Health Chatbot 🧠💬

MoodMate is a state-of-the-art mental wellness platform that combines advanced sentiment analysis with empathetic AI interactions.

## ✨ New Premium Features
- 📊 **Dynamic Mood Analytics**: Visualize your emotional journey with real-time charts.
- 🧠 **AI-Powered Sentiment Analysis**: Automatically flags urgent messages and provides personalized insights.
- 🔐 **Secure JWT Authentication**: Protected user data with encrypted passwords.
- 💬 **Conversational AI Assistant**: 24/7 support using OpenAI's latest models.
- 🎨 **Premium Aesthetic**: Responsive Glassmorphism UI with smooth animations.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Framer Motion, Chart.js, Lucide Icons
- **Backend**: Node.js, Express, MongoDB, Natural NLP, OpenAI API

## 🚀 Deployment (Vercel)
The project is pre-configured for Vercel deployment.
1. Connect your GitHub repo to Vercel.
2. Set Environment Variables (`MONGO_URI`, `JWT_SECRET`, `OPENAI_API_KEY`).
3. Deploy!

## ⚙️ Local Setup
1. `git clone https://github.com/MuhammadUmer2004-sys/Moodmate-Health-Chatbot`
2. `cd backend && npm install`
3. `cd ../frontend && npm install`
4. Create `.env` files based on `.env.example`.
5. Run `npm run dev` in both folders.
